# Account Setup

Learn how to set up and configure your Spotzy.ai account.

## Creating Your Account

1. Visit spotzy.ai and click "Sign Up"
2. Enter your email and create a password
3. Verify your email address
4. Complete your profile information

## Initial Configuration

After creating your account, you'll need to:

- Set up your organization details
- Configure billing information
- Add team members (optional)
- Set up your first phone number

## Security Settings

### Two-Factor Authentication

Enable 2FA for enhanced security:

1. Go to Account Settings
2. Click "Security"
3. Enable Two-Factor Authentication
4. Scan the QR code with your authenticator app

### API Keys

Generate API keys for programmatic access:

1. Navigate to Settings > API Keys
2. Click "Create New Key"
3. Name your key and set permissions
4. Store the key securely
