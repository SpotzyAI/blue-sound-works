# Creating Leads

Learn how to manually create individual lead records.

## Adding a Single Lead

### Quick Add

Add leads quickly:

1. Click "Leads" → "New Lead"
2. Enter required information
3. Add optional fields
4. Click "Save"

### Required Information

Minimum fields needed:

- **Phone Number**: Valid phone number
- **First Name**: Recommended
- **Last Name**: Recommended

### Optional Information

Additional fields:

- Email address
- Company name
- Job title
- Address
- Custom fields

## Custom Fields

### Creating Custom Fields

Add fields specific to your business:

1. Go to Settings → Lead Fields
2. Click "Add Custom Field"
3. Choose field type
4. Name and configure
5. Save

### Field Types

Available types:

- **Text**: Short text input
- **Long Text**: Multi-line text
- **Number**: Numeric values
- **Date**: Date picker
- **Dropdown**: Predefined options
- **Checkbox**: Yes/no values

## Lead Templates

### Creating Templates

Save time with templates:

1. Define common field sets
2. Pre-fill default values
3. Save as template
4. Use for quick creation

### Use Cases

Templates are useful for:

- Industry-specific leads
- Event attendees
- Webinar registrants
- Inbound inquiries
